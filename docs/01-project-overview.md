# Project Overview

The Smart Personal Finance Tracker is a web application designed to help users manage their personal finances in an easy and organized way.

The system allows users to track income and expenses, categorize transactions, and set monthly budgets. It also provides a clear visual overview of spending habits through charts and summaries.

This application is built to help students and individuals better understand their financial activities and improve budgeting habits. The frontend will be developed using React, while the backend will be implemented using FastAPI with a database for storing user and transaction data.
